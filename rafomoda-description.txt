<h3 id="item-description__rafomoda-responsive-fashion-landing-page-template">RafoModa – Responsive Fashion Landing Page Template</h3>
<p>
                <strong>RafoModa</strong> – Bootstrap 5 based Fashion Landing Page HTML Template. 
RafoModa is a clean, premium and modern HTML5 and CSS3 Template which can be used to create any site for multiple scenarios. 
It’s suitable for selling clothing, fashion, high fashion, men fashion, women fashion, accessories, digital, kids, watches, jewelries, shoes, kids, furniture, sports, tools.
<br><br>
Looks good on all types of screens devices and platforms (Android, iOS and Windows). 
This template is responsive on literally any screen size, this way you can worry less about interface, and focus on what is important. 
Use RafoModa, and create something amazing!
<br><br>
It has all the needed components for starting a brand new Fashion or redesigning your current website. 
It also has a online reservation form which is more it can be used. 
You can change all images and icons to make this template serve your business. 
Well documented, very easy to customize &amp; does not require much knowledge of coding.
